# Citation

If you use MCPower in your research, please cite it as follows:

## APA format

Lenartowicz, P. (2026). *MCPower: Monte Carlo Power Analysis for Complex Study Designs* [Computer software]. https://github.com/pawlenartowicz/MCPower

## BibTeX

```bibtex
@software{lenartowicz2026mcpower,
  author    = {Lenartowicz, Pawe{\l}},
  title     = {{MCPower}: Monte Carlo Power Analysis for Complex Study Designs},
  year      = {2026},
  url       = {https://github.com/pawlenartowicz/MCPower},
}
```

## Why cite?

Citing software helps:
- Other researchers find and evaluate the tools you used.
- Maintain transparency and reproducibility in your work.
- Support continued development of open-source tools.
